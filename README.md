# Sales Forecasting Using Machine Learning

## Project Overview

This project predicts future sales using historical sales data and machine learning techniques. A Random Forest Regressor model was trained on date-based features extracted from historical sales records.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* VS Code

## Methodology

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Performance Evaluation
6. Visualization

## Model Used

Random Forest Regressor

## Results

* Mean Absolute Error (MAE): 43.625
* R² Score: 0.9746

## Conclusion

The model achieved high prediction accuracy and successfully forecasted sales trends using historical data.
